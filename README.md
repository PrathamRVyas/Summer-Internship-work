# Internship-Chatbots

All the Major Internship Projects made at Rishabh Software

Three of the main projects are:

1. AI Interview Preparation Agent
This is a simple AI tool that helps students prepare for interviews

You enter your skills, domain, and experience level
The AI (Ollama + LangChain) creates technical, HR, and coding questions
It also gives sample answers and tells you what topics to improve
Built using Python and Streamlit for the UI

2. AI Resume Analyzer Agent
This project checks if your resume is good enough for job applications

You upload your resume as a PDF
The AI reads it and finds your skills using PyPDF2 + LangChain + Ollama
It gives an ATS score and shows missing skills
It also suggests small improvements to make your resume better

3. AI Project Idea Generator Agent
This tool helps students pick a final year project easily

You enter your skills, interests, and how much time you have
It gives 5 project ideas with tech stack and difficulty level
It also creates a simple week-by-week plan to build the project
Made using Python, Streamlit, LangChain, and Ollama
